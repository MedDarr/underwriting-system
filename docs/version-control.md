# Версионный контроль проекта

Для организации версионного контроля программной системы используется Git и удаленный репозиторий GitHub.

## Репозиторий

- Название репозитория: `underwriting-system`
- Владелец: `MedDarr`
- Ссылка: https://github.com/MedDarr/underwriting-system

## Назначение репозитория

Репозиторий предназначен для хранения исходного кода, документации, конфигурационных файлов, схем базы данных и материалов тестирования программной системы поддержки андеррайтинга.

## Инициализирующий коммит

В репозитории создана начальная структура проекта, включающая каталоги серверной части, клиентской части, документации, схемы базы данных и тестовых материалов.
